### **CLASE # 3**

**Tema:** Introducción a las Aplicaciones Informáticas y su Clasificación.
**Duración:** 45 minutos

**1. Objetivo de la Clase:**
*   Definir el concepto de aplicación informática y clasificarlas según su finalidad (ofimática, diseño, comunicación, etc.).
*   Diferenciar entre aplicaciones de software libre y software propietario, identificando ejemplos comunes de cada tipo.

**2. Método y Técnica:**
*   **Método:** Expositivo - Participativo
*   **Técnica:** Clase dialogada, contrastación de ejemplos y lluvia de ideas.

**3. Medios y Recursos:**
*   Pizarra o rotafolio.
*   Computadora y proyector (TV) para mostrar ejemplos visuales de diferentes aplicaciones.
*   Presentación digital con imágenes de interfaces de software (Word, Photoshop, Gmail, etc.).

**4. Desarrollo de la Clase:**

| **Etapa** | **Tiempo** | **Actividades Docentes y Discentes** |
| :--- | :--- | :--- |
| **Inicio** | **5 min** | **Repaso de la clase anterior:** "La vez pasada hablamos del Software de Sistema (S.O.) y de Aplicación. ¿Recuerdan la diferencia?" El docente recupera la tarea previa (ejemplos de software libre y propietario) y comenta algunas respuestas. |
| **Introducción** | **5 min** | **Orientación hacia el Objetivo (OHO):** "Hoy vamos a sumergirnos en el mundo de las aplicaciones que usamos a diario. Veremos para qué sirven y cómo se clasifican, no solo por su función, sino también por su tipo de licencia (libre o propietaria)." |
| **Desarrollo** | **25 min** | **<u>Bloque 1: ¿Qué es una Aplicación Informática? (5 min)</u>** <br> • El docente define una aplicación o programa como un tipo de software diseñado para realizar una tarea específica para el usuario (escribir, calcular, diseñar, comunicar). <br> • Se enfatiza que estas funcionan **sobre** el sistema operativo. <br><br> **<u>Bloque 2: Clasificación por Finalidad (10 min)</u>** <br> El docente presenta los tipos con ejemplos visuales proyectados: <br> • **Ofimática:** Procesador de texto, hoja de cálculo, presentaciones. (Ej: Word, Excel). <br> • **Diseño Gráfico:** Edición de imágenes y creación de gráficos. (Ej: Photoshop, Illustrator). <br> • **Comunicación:** Navegadores web (Chrome, Firefox), cliente de correo (Outlook, Gmail), mensajería (WhatsApp Web), redes sociales. <br> • **Audio y Vídeo:** Reproductores y editores. (Ej: VLC, Audacity). <br><br> **<u>Bloque 3: Clasificación por Licencia (10 min)</u>** <br> • **Software Propietario:** Se repasa el concepto. Se menciona que suele ser de pago y cerrado. **Ejemplos:** Adobe Photoshop, Microsoft Office, Windows. <br> • **Software Libre:** Se repasa el concepto (libertad de uso, estudio, modificación y distribución). **Ejemplos:** Navegador Firefox, editor de audio Audacity, suite LibreOffice. <br> • **Ventajas comparativas:** Se discute brevemente (costo, personalización, seguridad vs. estandarización, soporte técnico). |
| **Cierre** | **10 min** | **Conclusiones:** El docente pide a los estudiantes que nombren una aplicación que usen frecuentemente y que digan de qué tipo es (por finalidad y por licencia, si pueden). <br> **Tarea:** "Piensen en una actividad o hobby que les guste. Investiguen y anoten el nombre de una aplicación de software libre que se pueda usar para esa actividad." |

### **CLASE # 4**

**Tema:** LibreOffice: La Suite Ofimática Libre.
**Duración:** 45 minutos

**1. Objetivo de la Clase:**
*   Identificar LibreOffice como una suite ofimática de software libre y caracterizar cada uno de sus componentes principales y su utilidad.

**2. Método y Técnica:**
*   **Método:** Demostrativo - Explicativo
*   **Técnica:** Demostración práctica en vivo.

**3. Medios y Recursos:**
*   Computadora con **LibreOffice** instalado y conectada a un proyector (TV).
*   Pizarra o rotafolio para listar los componentes.

**4. Desarrollo de la Clase:**

| **Etapa** | **Tiempo** | **Actividades Docentes y Discentes** |
| :--- | :--- | :--- |
| **Inicio** | **5 min** | **Repaso y motivación:** "¿Alguien recuerda qué es una suite ofimática? ¿Y qué significa que sea de software libre?" El docente conecta las respuestas con la tarea previa. "Hoy vamos a conocer la alternativa libre más famosa a Microsoft Office." |
| **Introducción** | **5 min** | **Orientación hacia el Objetivo (OHO):** "En esta clase conoceremos LibreOffice en profundidad. Veremos cada uno de los programas que lo forman y descubriremos para qué sirve cada uno, preparándonos para usarlos en prácticas futuras." |
| **Desarrollo** | **25 min** | **<u>Bloque 1: ¿Qué es LibreOffice? (5 min)</u>** <br> • El docente lo define como un paquete de software de oficina libre, gratuito y de código abierto. <br> • Menciona que es compatible con los formatos de Microsoft Office y otros estándares. <br><br> **<u>Bloque 2: Componentes de LibreOffice (20 min)</u>** <br> El docente abre LibreOffice en su computadora y muestra cada aplicación, explicando su función principal: <br> • **Writer (Procesador de texto):** "Es para escribir documentos de texto, como cartas, informes o tesis. Es similar a Microsoft Word." (Abre un documento de ejemplo). <br> • **Calc (Hoja de cálculo):** "Se usa para trabajar con números: hacer tablas, cálculos, gráficos y análisis de datos. Es el equivalente a Excel." (Muestra una hoja con fórmulas simples). <br> • **Impress (Presentaciones):** "Diseñado para crear diapositivas con texto, imágenes y animaciones para exposiciones. Como PowerPoint." (Abre una presentación breve). <br> • **Draw (Dibujo vectorial):** "Herramienta para crear diagramas, flujogramas y dibujos simples." (Muestra su interfaz). <br> • **Base (Base de datos):** "Permite crear y gestionar bases de datos para almacenar y organizar información compleja. Similar a Microsoft Access." (Se menciona, pero no es necesario profundizar). <br> • **Math (Editor de fórmulas):** "Editor especializado para crear y editar fórmulas matemáticas." (Se menciona brevemente). |
| **Cierre** | **10 min** | **Conclusiones:** El docente hace preguntas de verificación: "Si tuviera que hacer un gráfico a partir de unos números, ¿qué componente usaría? (Calc). ¿Y si tuviera que hacer el trabajo escrito para otra asignatura? (Writer)". <br> **Tarea / Actividad sumativa:** "Averigüen y escriban en su cuaderno el nombre del componente de Microsoft Office que es equivalente a cada uno de los programas de LibreOffice que vimos hoy (Writer -> ?, Calc -> ?)." |
